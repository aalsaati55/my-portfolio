# AbdulMajeed AlSaati — Personal Portfolio

Personal portfolio website for **AbdulMajeed Ghassan AlSaati** — Cybersecurity & Full-Stack Developer based in Bahrain.

🌐 **Live site:** [aalsaati55.github.io/my-portfolio](https://aalsaati55.github.io/my-portfolio)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 + Vite |
| Styling | TailwindCSS v4 |
| Icons | Lucide React, Devicons CDN, Simple Icons |
| Contact Form | Formspree |
| Deployment | GitHub Pages (GitHub Actions) |

---

## Sections

- **Hero** — Name, animated typing role, social links
- **About** — Background, nationality, languages
- **Skills** — Interactive tech icon grid + categorized skill cards
- **Projects** — SentinelAI, Social Network, Netcat, Defense in Depth, GraphQL Dashboard, Localhost Web Server, wget
- **Education** — Bahrain Polytechnic, Reboot01, Al Rawabi
- **Certificates** — STRELKA, IELTS, BH National CTF
- **Contact** — Formspree-powered contact form

---

## Visual Effects

- Animated canvas particle network (200 particles, mouse repulsion, connection lines)
- Shooting star streaks
- 6 ambient glow orbs (purple & cyan)
- Dot grid background overlay

---

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
